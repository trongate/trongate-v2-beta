The Module Import Wizard

Trongate's module import wizard lets you steal features like a pirate.

  1. Copy a module.
  2. Paste it into your app.
  3. Activate.
  4. Done.

No package managers. No dependency chains. No third-party gatekeepers.

How It Works

Move features between apps like you're sharing photos:

  1. Grab module folder from App A
  2. Drop it into modules/ in App B
  3. Visit https://yoursite.com/module_name — SQL runs automatically

Got SQL tables? Drop .sql files in the module. The wizard handles them.

Child Modules? Same Deal.

Child modules import just like regular modules:

[code] modules/ shop/ product_reviews/ ← copy this entire folder Product_reviews.php views/ setup.sql ← SQL? Drop it here [/code]

Visit: https://yoursite.com/shop-product_reviews

The wizard detects child modules automatically.

Security First

Never run the wizard on production.

Import on localhost. Test thoroughly. Then deploy.

Tips

  • Module dependencies? Document them in a README.md inside the module
  • Config changes needed? Add instructions in INSTALL.txt
  • Assets? They're already in the module. Just works.

The wizard runs once per module. After setup, it disables itself automatically.
