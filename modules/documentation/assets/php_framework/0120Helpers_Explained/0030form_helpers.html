Form Helpers

Trongate includes a range of form helper functions designed to make form creation fast, intuitive, and reliable. The following functions are available from the Form Helper:

  • form_button() Generate a button element
  • form_checkbox() Generate a checkbox input
  • form_close() Close a form with CSRF token
  • form_date() Generate a date input field
  • form_dropdown() Generate a select dropdown menu
  • form_email() Generate an email input field
  • form_file_select() Generate a file upload input
  • form_hidden() Generate a hidden input field
  • form_input() Generate a text input field
  • form_label() Generate a label element
  • form_number() Generate a number input field
  • form_open() Open a form with specified action
  • form_open_upload() Open a form with file upload support
  • form_password() Generate a password input field
  • form_radio() Generate a radio button input
  • form_search() Generate a search input field
  • form_submit() Generate a submit button
  • form_textarea() Generate a textarea element
  • post() Retrieve form submission data
  • validation_errors() Display form validation errors
