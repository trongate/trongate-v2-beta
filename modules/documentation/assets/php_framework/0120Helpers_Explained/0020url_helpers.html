URL Helpers

Trongate includes a range of URL helper functions designed to make URL manipulation and navigation fast, intuitive, and reliable. The following functions are available from the URL Helper:

  • anchor() Generate an HTML anchor tag
  • current_url() Get the current page URL
  • get_last_segment() Get the last segment of the URL
  • get_num_segments() Get the number of URL segments
  • previous_url() Get the previous page URL
  • redirect() Redirect to a specified URL
  • remove_query_string() Remove query string from a URL
  • segment() Get a specific URL segment
