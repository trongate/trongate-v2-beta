Getting Connected

Trongate has been designed to interact with MySQL and MariaDB. Yes, other database types are possible. However, the core framework focuses on the web's most battle-tested databases.

The Database Config File

Everything starts in config/database.php.

Fresh install? You get this:

[code=php] [ 'host' => '127.0.0.1', 'port' => '3306', 'user' => 'root', 'password' => '', 'database' => 'trongate' ], ]; [/code]

Change the values. Hit refesh. Done.

Multiple Databases? Sure!

Trongate v2 lets you connect to multiple databases. Adding additional database connection groups is as easy as building onto a simple PHP array. For example:

[code=php] [ 'host' => '127.0.0.1', 'port' => '3306', 'user' => 'app_user', 'password' => 's3cureP@ss', 'database' => 'production' ], 'analytics' => [ 'host' => 'analytics.yoursite.com', 'port' => '3306', 'user' => 'reader', 'password' => 'R3ad0nly!', 'database' => 'stats' ] ]; [/code]

Controller files have the ability to query the default database only. Model files, on the other hand, can query any valid database group defined within config/database.php. For more details, please refer to our documentation on Model Files.
