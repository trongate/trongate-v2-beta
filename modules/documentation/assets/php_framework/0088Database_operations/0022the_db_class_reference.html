The DB Class Reference

Trongate comes with a fully featured PHP class for assisting with database interaction. The class is named 'DB' and is stored in a file named DB.php, inside the engine directory.

The DB class provides a comprehensive set of methods for working with MySQL and MariaDB databases. All methods can be invoked using $this->db from within your controllers.

In Trongate v2, use $this->db. If you see $this->model in reference materials, that's v1 syntax.

The DB class contains the following methods:

For Data Retrieval:

  • get()
  • get_where()
  • get_where_custom()
  • get_one_where()
  • get_many_where()
  • get_where_in()
  • get_max()

For Data Manipulation:

  • insert()
  • update()
  • update_where()
  • delete()
  • insert_batch()

For Data Analysis and Management:

  • count()
  • count_where()
  • count_rows()
  • describe_table()
  • table_exists()
  • get_all_tables()
  • resequence_ids()

For Custom Database Querying:

  • query()
  • query_bind()
