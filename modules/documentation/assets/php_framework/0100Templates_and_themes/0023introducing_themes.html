Introducing Themes

Templates are great. Themes are templates that come with their own motorcade.

A theme is a complete visual package: template file + CSS + JavaScript + images + fonts—everything needed for a consistent look, all living together in one folder under public/themes/.

Why Themes Rule

  • One folder = one complete design
  • Switch the entire appearance of your app with a single line
  • Color variations? Just sub-folders. No code changes
  • 100% portable—copy the folder to another project, register it, done
  • No framework lock-in. Use Bootstrap, Tailwind, plain CSS, or go full retro—your call

Themes Live Here

[code] public/themes/ default_admin/ blue/ css/ js/ admin.php ← the actual template file purple/ red/ bootstrappy/ dark/ css/ js/ images/ bootstrappy.php light/ my_cool_theme/ css/ js/ theme_template.php [/code]

Register Once, Use Forever

Add your themes to config/themes.php:

[code=php] <?php // config/themes.php $themes = [ 'admin' => [ 'dir' => 'default_admin/blue', 'template' => 'admin.php', ], 'admin_purple' => [ 'dir' => 'default_admin/purple', 'template' => 'admin.php', ], 'bootstrappy' => [ 'dir' => 'bootstrappy/dark', 'template' => 'bootstrappy.php', ], ]; define('THEMES', $themes); [/code]

Load Exactly Like a Normal Template

[code=php] $this->template('bootstrappy', $data); [/code]

Trongate checks if 'bootstrappy' exists in THEMES → loads the template from public/themes/bootstrappy/dark/ → sets the THEME_DIR constant to that path → your assets just work.

Asset Heaven with THEME_DIR

Inside any theme template:

[code=html] <link rel="stylesheet" href="<?= THEME_DIR ?>css/style.css"> <script src="<?= THEME_DIR ?>js/app.js"></script> <img src="<?= THEME_DIR ?>images/logo.png" alt="Logo"> [/code]

No path guessing. Works whether you're in blue/, dark/, or retro_8bit/.

The default admin area? Just a theme. Want a different color scheme? Change one line in config/themes.php:

'dir' => 'default_admin/purple'

Instant new look. Zero code changes.

One Line, Brand New Look

[code=php] $user_level = $this->session->get('user_level'); $theme = ($user_level === 'admin') ? 'admin' : 'bootstrappy'; $this->template($theme, $data); [/code]

Admins get one design. Regular users get another. Zero duplication.
