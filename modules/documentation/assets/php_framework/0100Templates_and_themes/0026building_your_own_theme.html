Building Your Own Theme

You've seen how themes load. Now make one. It's stupidly simple.

Step 1 – Choose a Theme Name

First, decide on a name for your theme. For this example, we'll assume a theme name of, "modern".

Your theme name must be a valid directory name. This means:

  • No spaces (use underscores instead: new_age not new age)
  • No special characters (stick to letters, numbers, and underscores)
  • Use lowercase for consistency

Good theme names: modern, admin_dark, new_age, retro_80s

Bad theme names: My Theme, super-cool!, theme#1

Step 2 – Create the Theme Folder

Create your theme folder inside public/themes/ using your theme name:

[code] public/themes/ modern/ css/ js/ [/code]

For this example, we're assuming that you have a CSS file and a JavaScript file prepared for your theme, hence the sub directories "css" and "js". In a real world example, you can have any filetype of your choosing added to your theme directory. This includes things like images, fonts and so on.

We won't bother giving you sample code for those files. The purpose of this page is merely to show you how themes get structured. It's for you to come with your own design ideas.

Step 3 – Create the Template File

Create public/themes/modern/modern.php:

[code=html] <!DOCTYPE html> <html lang="en"> <head> <base href="<?= BASE_URL ?>"> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <link rel="stylesheet" href="css/trongate.css"> <link rel="stylesheet" href="<?= THEME_DIR ?>css/modern.css"> <?= $additional_includes_top ?? '' ?> <title><?= $page_title ?? 'Modern Theme' ?></title> </head> <body> <div class="header"> <h1>My App</h1> </div> <div class="container"> <?= Template::display($data) ?> </div> <?= $additional_includes_btm ?? '' ?> </body> </html> [/code]

Step 4 – Register the Theme

Open config/themes.php and add:

[code=php] <?php $themes = [ 'modern' => [ 'dir' => 'modern', 'template' => 'modern.php', ], ]; define('THEMES', $themes); [/code]

The theme key ('modern') must match your theme directory name.

Step 5 – Create the Template Method

Add a method in templates/controllers/Templates.php and give the method the same name as your theme.

For example:

[code=php] public function modern(array $data): void { $data['additional_includes_top'] = $this->build_additional_includes($data['additional_includes_top'] ?? []); $data['additional_includes_btm'] = $this->build_additional_includes($data['additional_includes_btm'] ?? []); load('modern', $data); } [/code]

The method name must match your theme key from config/themes.php.

Step 6 – Use Your Theme

In any controller, load your theme using Trongate's template() method:

[code=php] public function index(): void { $data['view_file'] = 'welcome'; $this->template('modern', $data); } [/code]

Done. Your theme is live!
