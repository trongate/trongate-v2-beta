run()

public function run(): bool

Description

Executes the validation rules that have been previously defined using the set_rules() method. If all fields pass their respective validation rules, the method returns true. Otherwise, it returns false and populates error messages that can be displayed to the user.

Parameters

Parameter Type Description
None n/a This method does not accept any parameters.

Return Value

Returns true if all validation rules pass. Otherwise, returns false.

Example Usage

$this->validation->set_rules('username', 'Username', 'required|min_length[5]|max_length[20]');
$this->validation->set_rules('email', 'Email', ['required', 'valid_email']);

if ($this->validation->run()) {
  // Proceed with processing the data
} else {
  // Display validation errors
  $errors = $this->validation->get_errors();
}
