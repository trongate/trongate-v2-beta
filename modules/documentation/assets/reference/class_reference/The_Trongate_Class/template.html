template()

protected function template(string $template_name, array $data = []): void

Description

Renders a specific template view by calling a corresponding method in the Templates controller class.

Parameters

Parameter Type Description Default
$template_name string The name of the template method to be called. N/A
$data array Optional. An associative array containing data to be passed to the template method. []

Return Value

Type Description
void This function does not return a value.

Example Usage

The code below demonstrates how to render a template named 'admin'. In this example, as assortment of user data is being passed into the template, including some user details and a corresponding view file name.

[code=php] $data = [ 'first_name' => 'John', 'age' => 21, 'city' => 'London', 'view_file' => 'user_profile' ]; $this->template('admin', $data);[/code]
